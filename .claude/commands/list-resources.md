# List Available Resources

Browse what's available for picking.

## Steps

1. List each category:

```bash
echo "=== SLASH COMMANDS ==="
ls resources/slash-commands/

echo ""
echo "=== CLAUDE.MD TEMPLATES ==="
ls resources/claude.md-files/

echo ""
echo "=== WORKFLOW GUIDES ==="
ls resources/workflows-knowledge-guides/
```

2. Format as a clean organized list

3. End with: `Use /pick <category>/<name> to add a resource to your project`

## Filter

If argument provided, filter results to matching items only.

$ARGUMENTS
